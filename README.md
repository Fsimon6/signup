# 顺丰运费自动查询

自动从飞书文档表格读取「顺丰号」，在顺丰 IUOP 页面查询运费，并写回「实付顺丰邮费」列。

## 工作流程

```
飞书文档(顺丰号) → 复制单号 → 顺丰网页查询 → 获取交易金额（元） → 回填飞书(实付顺丰邮费)
```

## 安装

```bash
pip install -r requirements.txt
```

## 配置

复制 `.env.example` 为 `.env`，填写配置：

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `FEISHU_TAB_NUMBER` | `1` | 飞书文档所在的夸克浏览器标签页编号 |
| `SF_TAB_NUMBER` | `2` | 顺丰查询页面所在的夸克浏览器标签页编号 |
| `QUERY_DELAY_SECONDS` | `2` | 每条查询间隔秒数 |
| `SF_RESULT_WAIT_SECONDS` | `3.5` | 查询结果加载等待时间 |
| `MAX_ROWS` | `0` | 限制处理行数，0 表示全部 |
| `SKIP_FILLED` | `true` | 跳过已有「实付顺丰邮费」的行 |

## 使用前准备

1. 打开夸克浏览器，登录飞书账号
2. 在标签页 1 打开飞书文档表格（确保表格可见）
3. 在标签页 2 打开顺丰 IUOP 查询页面并登录：
   - `https://iuop.sf-express.com/#/ipcs/financeManage/transactions`
4. 确保浏览器窗口未最小化且可见

## 运行

```bash
python sf_automation.py
```

## 功能特点

- **智能列识别**: 通过文本识别自动查找「顺丰号」和「实付顺丰邮费」列
- **断点续传**: 自动记录已完成的单号，支持中断后继续
- **失败记录**: 查询失败的单号记录到 `.sf_failed.json`，方便后续处理
- **智能滚动**: 自动向右滚动查询结果区域，确保能看到右侧的「交易金额（元）」字段
- **防错机制**: 遇到无效单号或金额为0时自动跳过，不填写错误数据
- **纯GUI操作**: 不依赖浏览器控制台，避免安全提示拦截

## 故障排除

**未找到夸克浏览器窗口**: 确保夸克浏览器已打开，且窗口标题包含"夸克"

**未找到「顺丰号」列**: 确保飞书表格表头包含「顺丰号」文字

**输入框定位失败**: 确保顺丰页面已加载完成且处于登录状态，页面上能看到「顺丰单号」文字

**未提取到金额**: 查询结果可能加载较慢，尝试增大 `SF_RESULT_WAIT_SECONDS`
